Discord Bots

Language:Discord.js

On-going hobby: Started Summer 2018

The Story

School assignments are nice such as building a bank system, user profile, alarm clock, etc. But I like to code something that involves more user-interaction. I use discord a lot to talk to my friends and play games. So, I created a discord channel along with my discord bots. Let's see what my bots can do!

First Step

To develop a discord bot, I had to learn few things. - NodeJS Setup and Ready Event Setup.
After understanding the set-up, I decided to make a bot that can send welcome messages to new members who joins my channel. When it worked, I found it entertaining and wanted to make something more difficult.

Welcome Messaging Bot

Second Step

One of key feature in discord is having a role which constructs a hierarchy among members in the channel. Unfortunately, only the manager of the channel could assign roles to members. This created an inconvenience to regular members for not being able to get/remove roles when they wanted. I thought that this problem could be handled by using bot. Which turned out to be great and my channel member loves using it. Every year, I have been upgrading this system for my members to use it more easily.

2018 Version - Auto-role assigning by text: Some members does not know the exact name or command to execute the bot correctly


2019 Version - Auto-role assigning by emoji: Big gaps between each messages/taking too much space/repetition of words


2020 Version - Auto-role assigning by emoji/embed: Clean design/Shows all available roles in one embeded function

Third Step

I thought that people might need more entertainment in my channel. Most general thought that came through my mind was a game. Here, I created few games such as guessing numbers, rock-paper-scissors and a survival game.

Rock-Paper-Scissors-Lizard-Spock

- Simple game to play. Just type any of the 5 options after `!` and then the bot will pick their option immediately. Play quick for fun!
