Messenger App Project

Language:Kotlin/Android

User-Interface Project: Finished in December 2019

The Story

In this messenger application, there are many attributes that was taken into consideration under development. This application is developed through Android Studio which uses the programming language Java. There are pages like profiles, home page, login/register page, etc. Each page is linked to another page after a specific user activating an event from clicking a button. There were many factors in the GUI side to consider ensuring that the interface is easily understandable for the users to use without a manual.

I also used Firebase to connect to a Backend. Firebase started as a YC11 startup and grew up into a next-generation app-development platform on Google Cloud Platform.
